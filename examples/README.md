# WebSocket & Streaming Examples

This directory contains example clients for using the WebSocket and streaming endpoints.

## Files

- `websocket_client.py` - Python WebSocket client
- `websocket_client.js` - JavaScript WebSocket client
- `stream_client.py` - Python streaming (SSE) client
- `stream_client.html` - Web-based streaming demo

## Quick Start

### Python WebSocket Client
```bash
python examples/websocket_client.py
```

### Python Streaming Client
```bash
python examples/stream_client.py
```

### JavaScript WebSocket (Browser)
Open `websocket_client.html` in a browser.

### JavaScript Streaming (Browser)
Open `stream_client.html` in a browser.
